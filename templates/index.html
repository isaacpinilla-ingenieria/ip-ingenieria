Proceso de trabajo

Una forma clara de avanzar con cada proyecto

01

Recepción del requerimiento

Se revisa la necesidad del cliente, medidas, croquis, muestras, referencias o planos base.

02

Evaluación técnica

Se define alcance del trabajo, nivel de detalle, cubicación, tipo de plano y factibilidad de fabricación.

03

Desarrollo y entrega

Se generan planos, modelado, despieces o documentación según el proyecto y se coordina la entrega.

04

Apoyo para cotizar o fabricar

Se puede continuar con apoyo para cotización, listas de materiales o fabricación a pedido.
